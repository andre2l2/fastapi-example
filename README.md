# Fastapi example

## Run project

Create venv

```bash
python3 -m venv .venv
```

Active venv

```bash
. .venv/bin/activate
```

Install requirements

```bash
pip install -r requirements.txt
```

Run fastapi

```bash
fastapi run ./src/main.py
```

## Save requirements

For save of create requirements file

```bash
pip freeze > requirements.txt
```
